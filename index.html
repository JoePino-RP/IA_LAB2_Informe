Reconocimiento de Caractéres con Redes Neuronales

Luis Felipe Pinto Ortegón Luis Steven Parra Carlos Daniel Sanchez Delgado Joseph Enrique Rojas Pulido

Abstract

Neural networks are systems that allow information processing inspired by biological neural networks, in order to solve many complex tasks based on examples without requiring rules, that is, through a learning process. In this laboratory it is proposed to design and build a character recognition system based on neural networks, implementing a software for the creation and training using the Back-Propagation algorithm through the Toolbox of the MatLab software; In this software, the graphical interface was created, which allows interaction with the user. The practice is given in order to apply the knowledge imparted in the theory of neural networks focused on pattern recognition.

Posted by Parra, Pinto, Rojas, Sánchez - Abril 22, 2022

Resumen

Las redes neuronales son sistemas que permiten procesar la información, inspirados en el modelo biológico de los procesos que llevan a cabo las neuronas en el cerebro humano; con la finalidad de resolver tareas complejas basadas en ejemplos o muestras, siendo guiado por reglas o patrones de comportomaiento. En este laboratorio se propone diseñar un algoritmo capaz de identificar patrones basado en los principios y fundamentos matemáticos de las redes neuronales, implementando el lenguaje de programación Python, con las librerías Keras; haciendo modelos que se entrenen de forma manual y de forma automática. Adicionalmente se aplicará este algoritmo a una planta que consiste en un robot móvil con un manipulador

Abstract

In this laboratory it is proposed to design and build a Kohonen network, which will allow through unsupervised learning to find a discrete representation of the proposed space, as well as implementing the TSP or the traveler's problem to find the shortest path that can reach the neuron, trying different solutions to reach the most appropriate; in this software was made the graphical interface which allows interaction with the user. The practice is given in order to apply the knowledge imparted in the theory of neural networks focused on Kohonen networks and the traveler's problem.

Abstract

Las redes neuronales son sistemas que permiten el procesamiento de información inspirados en las redes neuronales biológicas, para resolver muchas tareas complejas a partir de ejemplos sin necesidad de reglas, es decir, a través de un proceso de aprendizaje. En este laboratorio se propone diseñar y construir una red de Kohonen, la cual permitirá a través del aprendizaje no supervisado encontrar una representación discreta del espacio propuesto, así como implementar el TSP o problema del viajero para encontrar el camino más corto que pueda llegar a la neurona. , probando diferentes soluciones para llegar a la más adecuada; en este software se elaboró la interfaz gráfica que permite la interacción con el usuario. La práctica se da con el fin de aplicar los conocimientos impartidos en la teoría de redes neuronales enfocada a las redes de Kohonen y el problema del viajero.
